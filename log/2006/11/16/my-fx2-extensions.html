我的Firefox 2擴充套件

很久以前有一篇我的Firefox擴充套件，紀錄了我在Firefox 1.x時所使用的擴充套件，那時候我是能不裝就不裝型的，不過最近升級2.0後卻開始猛裝，弄到現在總共裝了有26個，整理介紹如下：

  • All-in-One Sidebar 0.7 : 為了有像Opera那樣的方法開啟側邊欄才裝這套的，不過這套包了太多東西，希望有其他套件可以達到我想要的效果
  • Autohide 1.1.3 : 用來達到真正的全螢幕效果，如果要用S5之類的網頁簡報來作簡報的話不錯用
  • bbs2chreader 0.2.13 : 看2ch用的
  • Bloglines Toolkit 1.6.5 : 提醒我的bloglines上是否有新文章出現
  • Compact Menu CE 2.1.2 : 把選單工具咧藏起來
  • del.icio.us 1.2 : 方便我新增del.icio.us書籤
  • Download Statusbar 0.9.4.5 : 取代預設下載管理視窗用的
  • Fasterfox 2.0.0 : 似乎可以加速Firefox XD
  • FireBug 0.4.1 : 網頁除錯用，作網頁時的好工具
  • FlashGot 0.5.97.01 : 抓大檔時還是會用FlashGet
  • Gmail Notifier 0.5.6.5 : 提醒我Gmail是否有新信件
  • Google Toolbar for Firefox 2.1.20060807W : Google Toolbar
  • IE Tab 1.2.0.20061113 : 可以開IE分頁的套件，Windows only
  • Image Toolbar 0.6.4 : 會在圖片上浮出的小工具列
  • InfoLister 0.9f : 輸出這分清單用的擴充套件^^
  • Live HTTP Headers 0.13.1 : 檢視HTTP Header，個人興趣
  • Nightly Tester Tools 1.2 : 可以不管擴充套件的版本支援度，硬裝上去
  • Ook? Video Ook! 0.6.2 : 抓YouTube這類網站的影片用的
  • Right Encoding 0.2.2 : 右鍵選擇編碼
  • Pearl Crescent Page Saver Basic 1.3 : 將網頁存成圖片
  • Show Image 0.4.2 : 單獨重新讀取圖片
  • Skype extension for Firefox 2.2.0.44 : 裝完Skype 3.0 beta後就冒出來的XD
  • Tab Mix Plus 0.3.5 : 分頁瀏覽的擴充套件，內建的分頁瀏覽功能雖然已經加強不少，不過我還是覺得不夠
  • Talkback 2.0 : 內建的錯誤回報系統
  • User Agent Switcher 0.6.9 : 偽裝成其他瀏覽器:P
  • Web Developer 1.0.2 : 製作網頁的好工具
  • XHTML Ruby Support 1.4.2006100801 : 顯示Ruby標籤

最後就是今天開始用HEMiDEMi了，好像也有Firefox的擴充套件的樣子，可能也會裝吧。
