11月 25日

sitemap.xml

Google的sitemap.xml已經變成一個標準格式了，Yahoo和MSN搜尋都將支援此一種格式，以上是舊聞，這篇只是要說一下要到哪登錄自己的sitemap.xml，Google的話就是用他的webmaster tools，用Gmail帳號都入後就可以註冊自己的網址並加入sitemap的位置，不過之後還要自己去網頁加上認證碼，這樣Google才能確認這個網站是你所擁有的。Yahoo的其實也差不多，透過他的Site Explorer，在Feed的地方加上自己的sitemap.xml位置，一樣需要認證，不過是建立一個特定檔名的HTML（好像有兩三個方法），24小時過後就算完成了。至於MSN目前還沒有方法XD，可能要使用固定位置的sitemap.xml吧。

wordpress 2 升級的問題

如果升級後，他說你好像沒裝WP，那可以檢查一下資料庫中options資料表裡面的siteurl這筆資料是否有值，如果登入主控台看到以下的錯誤訊息：

Invalid argument supplied for foreach() in /home/waterblo/public_html/wp-includes/capabilities.php on line 19″

那檢查看看同個資料表的prefix_user_roles這筆資料裡是否有亂碼，當然那個prefix是代稱，預設安裝的話應該會是wp_user_roles。

HTML Work Group 再開

Tim Bernes Lee在10月27日發表了一篇Reinventing HTML，說到他們在推廣網路世界從HTML進到XHTML的這一步算是失敗了，所以他接下來打算組成一個新的HTML團隊，將會同時加強HTML和XHTML，並且會有和以往不同的成員。不過這個Group所製作的新的HTML和現在正在制定的XHTML 2將會是分開的兩個標準，所以在後續的討論中就有人持反對立場，認為已經這麼多標準了，而且事實上網路世界確實是有在朝著XHTML邁進的，為甚麼還要在增加一個標準？
