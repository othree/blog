01月 21日

Internet Explorer 7 build 5299 測試

昨天不小心弄到，理所當然的拿來做些測試，以下用條列的方式說明目前的測試結果

  • 可以設定多個Homepage
  • 新增tab快照的功能，可以看小圖選
  • 搜尋引擎不能排序，但是可以移除
  • 可以匯入匯出OPML
  • Tab最少一定會留下一個
  • 只有一個時快照按鈕就消失了
  • 背景圖fix正常了
  • pure css menu會有東西popup出來，不過定位還不太正確
  • box model預設（沒Doctype時）只修一半，padding向內算，border向外算
  • 加上<?xml version="1.0" encoding="UTF-8"?>也可以進入標準模式
  • application/xml+xhtml還是不正常，不是render得很慘就是會要下載
  • 有像Opera的放大縮小功能（連圖一起）

01月 16日

MacWorld Expo 06

這次的KeyNote我到今天才看完，大部分訊息這幾天已經傳得很廣了，我就不再覆述，只提幾個我覺得比較有趣的，首先是Intel版的OSX會有一個叫Rosetta的程式，用來在應用軟體完全轉換成Universal Binary（PowerPC和Intel CPU都可以執行的binary檔）之前，讓Intel CPU的Mac可以執行以前PowerPC用的應用軟體，在會場Steve就展示了Word和PhotoShop，的確都可以正常執行，效能看起來普普就是，不過有這個軟體，我想對於升級到Intel版mac的應用軟體支援問題都一掃而空了。

另外就是Steve Jobs真不愧是presetation之神，看了幾場KeyNote下來，每次串場的人（例如這次在講Office的那位）講起來感覺都差好多，我下週也有要上台講東西，不知道表現會如何XD。
