換系統?

因為Movable Type 3要收費了，所以我打算把這裡的系統換掉，最近就裝了Textpattern、pLog、Wordpress三套來測試過，算是各有優劣吧，其中Wordpress是我目前覺得最有趣的，一是它內建了textile 1.0、2.0b和Markdown三個plug-in，中文也有zonble大大幫忙翻譯。然後就是他的Exhibit這個Plug-in，讓我需要在文章內插入圖片變的非常容易，雖然還有一點小缺點，不過已經是非常好用了，幾乎一切動作都是real time的，應該是用了不少javascript，不過我只有在IE下測試，沒試過他在Mozilla的環境下能不能正常使用，我想過振子我就會決定要換到哪一個系統吧（最近在幫人寫PHP程式），然後就是一段艱苦的奮鬥了，因為我當初MT不是用MySQL當資料庫，然後現在幾乎所有的CMS資料轉換的程式都是在MySQL間互轉的...orz。

The Matrix Trilogy

駭客任務計畫推出10片裝的DVD BOX，分別的內容如下：

  1. The Matrix
  2. The Matrix Revisited （180分，同之前的花絮）
  3. Matrix Reloaded Extended Version （190分，多了55分的Enter The Matrix的畫面）
  4. Matrix Reloaded Revisited - Special Features （180分，有未公開的花絮）
  5. Matrix Revolutions
  6. Matrix Revolutions Revisited - Special Features （180分，和4差不多XD）
  7. The Animatrix （打成Animax....orz）
  8. The Roots Of The Matrix - Special Features （180分，我也不知是什麼:P）
  9. The Burley Man Chronicles - Special Features （75分，反正就花絮啦）
  10. The Zion Archive - Special Features （吉翁公國萬歲!!）

不過還不是確定的就是，預計的發售日期是12月12日，價格是45歐元（應該是歐元吧，又我一直不確定歐元和台幣的匯率），資料來源是Digital Spy，不過我是在Slashdot看到的就是。
