還等著做的idea

首先是利用javascript，讓使用者可以在select標籤中輸入自訂值，基本上我是構想用滑鼠的double click事件來觸發動作，不過因為對javascript+DOM還不熟析，所以一直沒動作。第二個已經被Google X做走了（不過Google X消失了@_@），而且是我想到這個idea後隔一天就...。所以我決定做個簡單的mouseover放大效果，大概可以用在一些文字連結、導覽列之類的地方，當然這個是CSS only，而且不影響排版:P，現在在思考要做成什麼樣子，因為這不會很複雜，大概有空就會弄了（可惜研究生很沒空...）。

03月 06日

Netscape Browser 8.0 beta

終於出來可以下載了，我稍微試用了一下，發現，這次沒有收信軟體，才注意到名字叫Netscape Browser，其他方面表現都還不錯，像是個別網站的信賴度設定就很方便，而且tab browse的功能也比firefox好，也有快速搜尋的欄位（不過剛裝好只有netscape搜尋可以用，然後我沒辦法安裝其他引擎上去，雖然看起來可以切換），大部分的設定畫面都和Firefox很像，可以容易上手，不過extension目前還沒有共通就是，試著安裝都說會和現在的Netscape不相容，所以不能安裝。至於我最期待的IE和Firefox（其實程式裡稱為Netscape）兩種引擎的選擇，果然不辜負我的期望，應該是忠實的實現了兩種瀏覽器的結果了吧，不知道IE VIEW是怎麼弄出來的，不管如何，這個功能做的蠻好的，和他獨有的site control整合在一起，site control除了可以弄至每個網站的rendering engine要用哪個外，還有網站的信賴度，要不要開啟javascript等等，這功能我覺得作的很不錯，其實如果Netscape 8做的好的話（現在還是beta），要勝過firefox不無可能，不過希望他們能加入session saver之類的，不然現在是每次重開瀏覽器，網頁都要重連的狀況。

03月 05日

sIFR

這兩天又發現這個東西，目的和A List Apart的Dynamic Text Replacement一樣，不過我覺得sIFR強很多，因為Dynamic Text Replacement所做出來的文字是圖片，但是sIFR的卻還是文字，因此你可以選取複製，這點在親合力上實在好很多，我看了一下使用方法，似乎是把字型檔放進flash的檔案裡，然後再用javascript做處理，這方面我就不是很了解了，不知道到底是怎麼辦到的@_@。另外還有一個sIFR wiki，有興趣的人也可以去看看，說明文件和FAQ都在這，我自己是還沒測試過，因為沒有flash也不能新增字型，不然我還蠻想試試看中文有沒有用的:P。

03月 01日

XMLHttpRequest

最近注意到這東西，一開始是在Ajax: A New Approach to Web Applications這篇文章裡看到的（因為榮登某日的del.icio.us榜首，我就點過去看了），雖然我還沒有詳細看內容，不過照各方的介紹來看，這東西發展淺力很大，簡單用一句話來說，就是「update without refresh」，上面還說Google Maps和Gmail都是用這技術做的（題外話，gmail似乎有中文介面了，不過現在好像沒辦法自己調整），其實我本來都只知道gmail是在iframe裡面讀取資料，沒想到他是用這個技術，我現在則是在想不知道我的研究能不能用上這個東西，其實我現在是很想玩玩看這東西啊。
