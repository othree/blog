08月 13日

我的Firefox擴充套件 加強版

因為有大大說要詳細介紹，所以有了這篇文章。

  • Bloglines Toolkit Bloglines官方的擴充套件，主要功能有二，通知是否有新文章和右鍵選單的快速訂閱功能，不過我只有用前面的功能順便當書籤XD。
  • Download Statusbar 我想有不少人對於Firefox自動跳出的下載視窗會感到厭煩吧，不過要是關掉又完全看不到下載狀況，所以就有了這個擴充套件，當你下載檔案時，他會在視窗下面新增一個工具列，在上面顯示下載的狀況，如此不但不會影響瀏覽，還是可以掌握下載的狀況。而最近的最新版還有mini mode可以融入到原本就有的狀態列。
  • Gmail Notifier 這個真的不用說了吧XD。
  • Link Toolbar 支援HTML的link標籤，會在狀態列右方顯示一系列按鈕。
  • Tab Mix 雖然用來增強標籤頁瀏覽的擴充套件不少，不過我最後還是選擇這台灣人開發的，主要原因有三：1.設定簡單明瞭。2.內建session saver。3.可以讓javascript開啟的小視窗不會被開到標籤頁（其他擴充套件我不知道是辦不到還是我不會設定）。
  • Web Developer 有在製作網頁的人會用到的很多功能都包含在裡面，像是調整視窗大小、即時編輯CSS、關掉javascript...等等，不過我是只有要用時才讓他顯示出來。

另外再多介紹一些以前有裝覺得不錯的擴充套件（這部份應該還會繼續更新）：

  • XHTML Ruby Support 讓Firefox支援XHTML的Ruby標籤，這是用來標音用的標籤，IE已經支援好一陣子了。
  • 0rz.net 快速建立短網址用的。
  • ScrapeBook 抓網頁用，很方便使用，還可以作註記，畫重點。我是在學校的電腦才有裝，主要是用來抓要列印的部分（可以抓整頁也可以抓部分），然後再列印的，比較高級的用法是用來作個人資料的收集和整理。
