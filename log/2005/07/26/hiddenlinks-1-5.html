Hidden Links 1.5

之前的隱藏不必要的連結的小script更新了，現在支援IE/Win 5.5以後、Firefox、Opera、Safari，出現方式由原來的放大縮小改成上下移動加淡入淡出，不過淡入淡出效果只有Firefox和Safari支援，Opera似乎因為CSS 3還未正式定案所以沒有支援opacity屬性，想測試的可以去測試網頁右上角20px的範圍內停留一下。

想使用的人需要注意的是，因為我使用了SixApart的yDSF，所以要先去抓yDSF要用的兩張圖片和css檔並讓網頁連結它們，然後還要對它的CSS做點修改，避免在safari上產生顯示問題（不在意的就不用改了）。找到以下的CSS：

.ydsf:after {
	margin: -10px 0 0 -10px;
	background-position: left bottom;
}

改成

.ydsf:after {
	position: absolute;
	left: 0;
	bottom: 0;
	margin: -10px 0 0 -10px;
	background-position: left bottom;
}

最後下載我的兩個檔案：CSS、Javascript，一樣讓網頁連結到他們就可以了。
