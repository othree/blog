Yahoo拍賣資料外洩

沒想到我自己接到詐騙電話了，上網查一下看來不少人遇到過，不過我想這消息越多人知道越好，所以還是說一下。

對方有大陸口音，有口音的電話我都會覺得怪怪的，雖然聽說現在詐騙集團已經有正音訓練了，不過我好像還沒遇過訓練過的，他會自稱是賣家先跟你確認你是否在Yahoo拍賣上作過某筆交易，然後說因為他們員工的操作錯誤所以可能會讓你連續扣款12個月，要跟你確認是不是要取消這個設定，而且已經是取消期限的最後一天了。不過我跟他對話到這邊就說我要確認過後在取消了，因為像這種分期付款一定事由我們這邊來作確認，不可能對方按幾下就可以一個月扣走我一些錢，然後我說要在確認過後對方似乎就很失望的結束電話，之後他會說什麼我也不知道。

Yahoo外洩的就是交易資料和買賣家的聯絡資料，所以詐騙集團除了交易的內容外，還能取得我的手機號碼。然後我剛剛又接到自稱家樂福的，不過不是打手機而是打市話了。

IE的png gamma解讀錯誤應對方法

雖然最新的IE 7終於支援半透明png了，但是gamma的錯誤判斷卻還是沒改正，會讓圖片顏色和實際的有誤差，今天在作網站改版的時候順便找了一下有沒有辦法解決，結果找到Fixing the PNG Color Problem in IE這篇文章，介紹了TweakPNG這個小軟體，用這個小軟體開啟png圖檔後，把裡面的gAMA屬性刪除後重新存檔就可以了，雖然是辛苦作網站的人啦。

JavaScript 的 false

剛剛看到JavaScript quirks這篇文章，提到了 JavaScript 的 false 在做邏輯判斷的一些行為，首先就是拿 false 和 0 以及 空字串 作是否相等的運算都會回傳 true，但是和 undefined 以及 null 來判斷卻是相反地結果，這四種資料在直覺上都應該是 false 才對，不知道有沒有人知道背後的原因。

Safari on Windows

現在正在進行的WWDC公佈了這項新消息，今天就有測試版可以下載。

簡單測試報告來了，一部分我自己的，一部分是其他人的，首先，英文字畫得很漂亮，內建字型，使用者介面也很像OSX，整體感覺比iTunes還更像OSX的應用程式，包括選單文字的去鋸齒、設定視窗會動態調整大小，還有Google的作業系統判斷都會把他當成是Mac OSX，目前比較大的問題有幾個：ㄧ是使用書籤功能可能會當掉，位置列不太能輸入，中文輸入也不正常，亞洲文字（中文、日文）會掉字，而且還有分掉很多和掉很少兩種，我是屬於掉很多那種，整個網頁只剩幾個字這樣，而big5編碼的網頁中文字整個畫亂掉，話不多說，來看些圖片吧。

補充：剛剛看到有人說要有正黑體才會正確顯示中文，所以換一台有正黑體的電腦測試後果然正常許多，中文字型有缺些字是用新細明體來畫（不過也有人是這個字就不見了），日文的話則是沒有這種問題，圖片也補上了。
