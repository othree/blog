CSS Mastery 中文版

蠻可惜的，這是一本內容不錯的書，不過卻敗在翻譯、排版和印刷品質之下，首先翻譯，關於這點我不敢說太多，因為我自己文筆不好又沒什麼翻譯能力，但是esc字元和跳脫字元出現在同一章只隔幾頁的地方似乎也說不過去，另外一個問題就是不少地方我會好奇他原文是什麼，甚至在猜他是不是有翻錯。

排版的主要問題在程式碼的部份，是整個灰底來和本文作區別，但是因為印刷品質不是很好，紙也用得很普通，感覺對比不夠強，邊界也不夠清晰，當然這見仁見智啦，畢竟不影響到閱讀，應該也有人覺得根本就沒這問題吧，畢竟我是和一些進口書比就是了。最後是印刷品質，這就直接看圖吧，希望我這本是例外啦。

至於內容的部份，就如書名所說的，這本書不是說CSS是什麼、有幾種指定方法、有哪些屬性可以用，而是在知道這些基本知識後，還有什麼可以學的，像是可能會遇到的問題、如何除錯、影像取代法、css hack等等。雖然是我或多或少都有看過，但是整本書看完後好像又有什麼領悟，回來修改這裡的HTML和CSS，結果又弄掉不少多餘的HTML標籤，我來整理的話這本書大概有以下幾個重點：

  • 結構更好的HTML，不過著墨不多，我自己也等著看HTML Mastery。
  • 常見的CSS問題，像是IE的box modeling。
  • CSS除錯方法。
  • CSS hack，雖然我不太喜歡這東西，不過這裡講的很精要，有點像是剛好夠用的程度。
  • 圖片的使用，大部分是關於背景圖可以做到怎樣的應用。
  • 最後的兩個專案研究。

單就內容來說我覺得絕對值得一看，不過如果長期有在外國的相關網站注意各種新技巧的話，這本書的可看性就會小很多，因為裡面很多的東西都是網路上找的到的，其實感覺上像是近年流行的CSS技巧整理，至於要不要買中文版就只能看個人啦。
